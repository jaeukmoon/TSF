Run `python run.py --model ` — writes `results//ltsf/*.json` 3. Run `python leaderboard.py` — regenerates the static fallback [leaderboard.md](leaderboard-static.html) 4. Edit the `DATA` block in `leaderboard.html` to include the new row (automation TODO) For published (non-harness) numbers, use the [`tsf-scout`](https://github.com/jaeukmoon/TSF/blob/main/.claude/agents/tsf-scout.md) agent — it sweeps arXiv + updates [SOTA 2026](docs/sota_2026.html).
